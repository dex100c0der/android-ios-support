Smali Patcher

Smali patcher is a utility program designed to patch services.jar on an android system to spoof location, disable signature verification, signature spoofing and allow screenshots in secure apps. It is developed by XDA user fOmey. This tool works on all versions of Android and all android phones.

PATCHES:

  • Mock locations - Hide mock locations status, allowing apps like Pokemon GO to treat them as genuine location updates.
  • Secure flag - Allow screenshots/screensharing in secure apps.
  • Signature verification - Disable signature verification allowing modification/execution of signed system apps.
  • Signature spoofing - Enable signature spoofing app permission.
  • Magisk reboot (Galaxy S10) - Reboot directly back into magisk from powermenu.
  • High volume warning - Disable high volume popup dialog.

NOTE:

GPS SPOOFERS / FUSEDLOCATION / RUBBERBANDING:
FusedLocation API GPS spoofers are NOT compatible with the mock location patch.You are required to use a traditional GPS spoofer! Spoofed location updates will be fed into fusedlocation all the same, meaning you don't need to disable the service.. which means while your not spoofing you still gain the benefit of fusedlocation which is all around more accurate location updates!

REQUIREMENTS:

  • PC with at least 3GB+ of RAM.

  • Framework 4.7.1 or later installed.

  • Java installed.

  • Android device running 6.0 or above.

INSTRUCTIONS (ADB METHOD/AUTOMATED - RECOMMENDED!)

  1. Backup device.. safety first kids.
  2. Enable USB debugging in DEVELOPER SETTINGS on your device and connect via USB to your PC.
  3. Run "SmaliPatcher.exe", First startup will automatically download the latest necessary binary's.
  4. Select your desired patches, but certainly check [Mock locations].
  5. Hit "ADB PATCH" button.
  6. Once the process has completed, magisk module will be generated in the same directory as "SmaliPatcher.exe" named: "SmaliPatcherModule-X.X.X.X-fOmey@XDA.zip".
  7. Install the generated zip file as a module in Magisk Manager.
  8. Reboot after that.
