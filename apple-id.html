HOW TO CREATE NEW APPLE ID

/discord/ ipogo

  • Must be done on a computer running Windows operating system (to have no 2FA security).
  • Apple ID ending with "@icloud.com" cannot be used.

    • STEP 1:

    Download and install iTunes (Win64-bit) (Win32-bit) on your computer.

    STEP 2:

    Open iTunes > click Menu > Sign In > Create New Apple ID.

    STEP 3:

    Fill up the registration form, faking all other information (up to you) except for the e-mail address.

    STEP 4:

    On verification: choose Phone > enter a valid phone number > select Text Message. Verify the phone number you provided by entering the code you received (via text) in the textbox window.

    STEP 5:

    After successfully creating a new apple ID, sign out of iTunes and sign into iTunes with your iDevice apple ID.

    STEP 6:

    Use the newly created apple ID now for installing IPA.
